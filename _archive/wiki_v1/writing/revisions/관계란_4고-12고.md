# 관계란 — 버전 비교 (4고 ↔ 12고)

> 작품 카드: [[wiki/writing/plays/관계란/00_작품카드_draft]] · 공통 주제: [[wiki/shared/themes/제도와 인간]]
> ⚠️ 4고와 12고(양 끝)만 정독. 5~11고의 중간 분기는 미확인. 아래는 *방향성*이지 단계별 경로가 아님.

## 한눈에 보는 변화

| 항목 | 4고 | 12고 |
|---|---|---|
| 인물 표기 | **A / B** | **ㄱ / ㄴ** (더 기호화·익명화) |
| 과거 장면 수 | **5개** (권태기·첫만남·전화·썸·두번째만남) | **3개** (첫만남·전화+관계질문·헤어짐) |
| 과거 배열 | 현재1 뒤 **과거 블록**(권태기→첫만남… 비순차) | 현재/과거 **교차**, 과거는 **순차**(만남→연애→이별) |
| 사랑 고백 | B가 **"사랑해" 직접 발화** | "당신 입에서 듣고 싶어" — **말해지지 않음** |
| 권태기(아침식탁·약·우산) | 있음 | **삭제** |
| 헤어짐 장면 | 없음(연애 중반에 머묾) | **신설**("나는 계속 없는 사람 같아요") |
| 현재 "제가 보호자입니다" 정면 대치 | 없음 | **신설**("사랑했어요"—"법적 관계가 아닙니다") |
| 결말의 '단 한 번' 순간 | **시각**(B 얼굴에 빛이 들었다 꺼짐) | **청각**(ㄴ 목소리 "이름 같은 거 있으면") |

## 1. 구조

- **4고 = 현재 틀 + 과거 블록.** 현재1(응급실) 직후 과거 5장면이 *한 덩어리*로 이어지고(권태기→첫만남→전화→썸→두번째만남), 그 뒤 현재2(복도)·현재3(창구)로 닫힌다. 과거의 시간이 **비순차**다 — 권태기(관계 말기)로 먼저 열고 첫 만남으로 거슬러 오른다.
- **12고 = 현재/과거 정밀 교차.** 현재1→과거1→현재2→과거2→현재3→과거3→현재4. *한 칸씩 번갈아* 들어가고, 과거는 **순차**(만남→연애→이별)로 정렬돼 현재(입원→배제 심화)의 진행과 **나란히 상승**한다.
- (AI 제안) 변화의 핵심은 *몽타주의 규율화*다. 4고가 "현재에서 과거를 회상하는" 구조라면, 12고는 "두 시간이 같은 속도로 교차 상승하는" 구조 — 마지막 현재4(관계란)와 과거의 종착(헤어짐)이 **같은 무게로 충돌**하도록 설계됨.

## 2. 인물성 (욕망과 말투)

- **표기 A/B→ㄱ/ㄴ**: 고유성이 더 지워짐. 이름이 없는 관계라는 주제가 *인물 명명 단위*에서부터 수행됨.
- **ㄴ(연인)의 욕망 표현이 절제됨**: 4고는 "사랑해"를 *입 밖에 냄* → 12고는 "당신 입에서 듣고 싶어"로 *상대에게 떠넘김*. 사랑이 **발화에서 결핍으로** 이동. ㄴ의 핵심 결핍("나는 계속 없는 사람 같아요")이 12고 헤어짐 장면에서 **명시적 대사**로 승격.
- **ㄱ의 수동성 → 능동의 한 순간**: 12고는 현재3 "제가 보호자입니다 / 사랑했어요"라는 *정면 시도*를 신설. 4고의 ㄱ은 창구 앞에서 더 절차적·수동적이었는데, 12고는 한 번 *밀어붙였다가 기각*당하는 곡선을 추가 → 배제의 잔인함이 더 선명.

## 3. 장면 리듬

- **4고**: 과거 블록이 길어 *연애의 결*(권태·썸·전화의 머뭇거림)을 충분히 깐다. 호흡이 상대적으로 완만, 정서의 축적형.
- **12고**: 장면이 짧아지고 침묵·말줄임이 더 촘촘. 현재와 과거가 빠르게 부딪혀 *대비의 충격*이 커짐. "배우자—자녀—부모—형제자매—"의 **목록 붕괴**가 두 버전 모두 있으나, 12고는 군더더기를 덜어 그 반복이 더 도드라짐.
- (AI 제안) 4고는 *관계의 두께*, 12고는 *배제의 날카로움*에 리듬을 맞췄다.

## 4. 상징의 선명도

- **관계란(서류 칸)**: 두 버전 공통의 중심. 12고는 곁가지(권태기 소품들: 우산·약)를 쳐내 *서류·창구·유리*에 상징을 집중.
- **'단 한 번' 순간의 매체 전환**: 4고는 **시각적**("B의 얼굴에 처음으로 빛이 든다 … 빛이 다시 꺼진다") — 제도 뒤에 사람이 있음을 *보여줌*. 12고는 그 시각 연출을 빼고 **청각적 메아리**(ㄴ "이름 같은 거 있으면", 흩어지듯)로 닫는다 — *들려줌*.
  - ⚠️ 작품 카드의 "단 한 번 빛이 돌아오는 순간(현재 창구)" 서술은 **4고에 더 정확**. 12고는 청각으로 전환됨 — 카드 표현은 *모티프 차원*으로 읽되, 12고 한정 서술은 보정 필요(Lint 항목).
- **손짓의 회귀**(과거 연인의 "네"+손을 테이블에 놓기 ↔ 현재 간호사가 서류를 내려놓기): 4고 일러두기·지문에 *명시적*("…똑같은 방식으로. B는 모른다"). 12고는 이 장치를 지문에서 덜 설명 → 상징을 *숨김*.

## 5. 설명 과잉 여부

- **4고**가 더 설명적이다: 일러두기에 "단 한 번 빛이 돌아오는 순간이 있다", 지문에 "첫 만남에서 B가 '네'라고 답하며 손을 테이블에 놓던 것과 똑같은 방식으로. B는 모른다", 대괄호 [ ] 연출 지시("관객에게 전달되지 않는다") 등 *메타 지시*가 많다.
- **12고**는 그 설명을 상당수 걷어냄 → 관객이 *눈치채야 하는* 구조로 이동. (AI 제안) 신뢰의 이동이다: "연출/배우에게 설명" → "관객에게 맡김". 단, 손짓·빛 회귀 같은 *섬세한 호응*이 설명 없이 전달될지는 무대 검증 필요.

## 6. 무대성

- 두 버전 모두 **빈 무대 + 의자 둘 + 창구**라는 미니멀 구성은 동일.
- 12고는 *조명·소음의 층위 분리*(현재=흰빛/소음, 과거=따뜻/무음)에 더 의존 — 장면 전환을 대사·소품이 아니라 **빛과 소리**로 처리. 배우 1인 2역(ㄴ=연인/제도)의 *전환 연기*가 더 큰 무대적 과제로 부상.
- (AI 제안) 12고의 "병원 소음이 한 박자 꺼진다. 텅." 같은 *음향의 구두점*이 장면의 골격을 짊어진다 — 음향 디자인이 거의 또 하나의 배우.

## 7. 이전(4고) 버전에서 사라진 강점

- **권태기 장면의 상실**: 4고의 아침식탁("약은 확인했어?", "우산 못 봤어?", 끝까지 눈 안 마주침)은 *함께 사는 관계의 소진·무명(無名)*을 보여주는 결이 있었다. 12고는 이를 버리고 *사랑–이별–배제*로 압축 → 강렬해진 대신, **"일상의 관계도 이름이 없다"**는 또 다른 층이 약해짐.
- **B의 직접 고백("사랑해")**: 사라지며 절제가 강해졌지만, *한 번이라도 말해진 사랑*이 주는 온기/대비도 함께 사라짐.
- (AI 제안) 12고가 얻은 것(날카로움·구조적 긴장)과 잃은 것(관계의 두께·온기)은 트레이드오프. **권태기 한 조각의 부분 복원**은 검토 가치(작품 카드 '수정 우선순위'와 동일 제안).

## 8. 다음 수정 우선순위 (AI 제안 — 사용자 의도 우선)

1. **결말 '단 한 번' 순간의 매체 결정**: 시각(4고)·청각(12고) 중 무엇으로, 혹은 둘을 어떻게 겹칠지 확정. 작품의 정점이므로 1순위.
2. **권태기 결의 복원 여부**: "일상의 무명"을 한 장면이라도 살릴지 — 살린다면 *짧게*, 현재의 배제와 운율이 맞도록.
3. **손짓 회귀의 가시성**: 12고에서 설명을 뺀 손짓 호응이 무대에서 읽히는지 리딩/리허설로 검증. 안 읽히면 *최소 신호*만 복원.
4. **5~11고 ingest**: 중간 개작에서 *언제 권태기가 빠지고, 언제 헤어짐이 들어왔는지* 분기점 확인 → 이 비교의 빈 구간을 메움.
